# Volunteer Center Database Redesign

## Overview

Redesigned the Volunteer Center’s Salesforce HandsOn Connect platform to improve usability, accessibility, and mobile experience for a large community-based organization supporting over 300 nonprofit partners.

## Problem

The existing volunteer portal had several key issues:

* Outdated UI/UX compared to the main website
* Accessibility challenges (poor contrast, readability, navigation)
* Weak mobile experience despite ~70% of users accessing via mobile devices
* Inconsistent branding across platforms

## My Role

* Conducted system analysis of the Salesforce HandsOn Connect platform
* Identified UX, accessibility, and workflow inefficiencies
* Collaborated with stakeholders to define requirements and improvement areas
* Developed redesign recommendations focused on usability and accessibility

## Key Improvements

* Proposed UI/UX redesign aligned with modern design standards and organization branding
* Improved navigation structure and user workflows for easier volunteer registration
* Applied WCAG 2.1 accessibility principles (contrast, readability, usability)
* Recommended mobile-first design improvements for better cross-device experience

## Results

* Targeted **25% increase in volunteer registrations**
* Targeted **30% improvement in navigation efficiency (task completion time)**
* Enhanced accessibility and usability for a diverse user base
* Streamlined system usability for both volunteers and internal staff
